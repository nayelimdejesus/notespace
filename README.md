# Notespace

Notespace is a personal journaling and mood-tracking web application that offers users a private, reflective space to record thoughts, secrets, and daily experiences. It includes an interactive mood visualization feature that tracks emotional patterns over time using a color-coded chart similar to GitHub's contribution view.

## Core Features

- Secure Journal Entry

- Mood Tracking via Color Chart

- Mood to Entry Linking

- Authentication & User Management

- Search & Filter

## Technical Stack

Backend: Django, PostgreSQL

Frontend: HTML, CSS, JavaScript with Django templating
Forms: Django Forms for entries and mood selection
